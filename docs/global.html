AVObjMacro(macroName, operationBeforeopt, operationAfteropt)

Command for extending the AVObj.extend method with additional macros. The operationBefore function has two arguments passed in when called:
  1. properties - The new object's properties before it's created.
  2. options - whatever was passed into the macro when defined.
After the object is constructed the operationAfter is fired. The operationAfter function has two arguments passed in when called:
  1. object - The object that was just constructed
  2. options - whatever was passed into the macro when defined.

Name Type Attributes Description
macroName string The name of the new macro you want to create.
operationBefore function <optional>
 The function to call when the macroName is defined.
operationAfter function <optional>
 The function to call when the macroName is defined and the class object has just been constructed.
