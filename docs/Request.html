Class: Request

Request

The Request class is used for creating ajax page requests. Note that once the Request has been complete, the object will self destruct as soon as the requested data is returned.

Request.create(method, url, asynchronous, finishFunction, data)

Call Request Init to create a request for a specific page.
Parameters:
Name Type Description
method The way the data should be sent. (Either Post or Get).
url The path to the file you want to load.
asynchronous If the event should halt all other processes.
finishFunction The function that will fire once the call is completed.
data Additional info such as form data goes here.
